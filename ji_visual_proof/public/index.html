A Visual Proof of Jensen's Inequality

Here I give a visual proof of Jensen's Inequality, using convex analysis.

Shown below is a convex function and the convex hull of its plot.

In the above, let f(x) be a function defined on domain X, and let f(X) be the image of f in the same order as elements in X. By the definition of convex function, every point in the hull lies on or above the plot itself. Also by definition of convex hull, every point in the hull consists of some convex combination w of all of the points (x, f(x)) in the plot itself.

The points in the hull are every possible convex combination w of the set of plot points. The key thing to notice is that, for any point in the hull (i.e. any choice of w), there exists a point in the plot that is equal or directly below it. Using this pair of points, we can write:

\begin{aligned} w \cdot X &= x && \text{x exists for this choice of w} \\ f(x) &\le w \cdot f(X) \\ f(w \cdot X) &\le w \cdot f(X) && \text{Jensen's Inequality} \end{aligned}

which is true for any choice of w. The final result is Jensen's Inequality. It says that a convex combination w of the function values (in domain order) will be not less than the function value of that same convex combination of the points in the domain.
